## Welcome to Pulse

Pulse is a modern productivity application built with clean OOP architecture and the latest technologies. It features a complete **idea management system** with collaborative editing, project organization, and real-time synchronization powered by **Convex**.

## üöÄ Key Features

- **Clean OOP Architecture**: Separation of concerns with adapters, services, and repositories
- **Real-time Collaboration**: Multi-user editing with Convex and ProseMirror sync
- **Type-Safe**: Full TypeScript support with strict typing throughout
- **Modern UI**: Built with React, TanStack Router, Tailwind CSS, and Flowbite
- **Scalable Backend**: Convex serverless functions with real-time database
- **AI Integration**: OpenAI GPT integration for enhanced productivity

- **Documentation**: Starlight (Astro-based documentation)
- **Development**: Turborepo monorepo with pnpm workspaces

## ü§ù Contributing

This documentation helps developers understand and contribute to the Pulse productivity application. Whether you're implementing new features, refactoring services, or maintaining the OOP architecture, you'll find comprehensive guides and examples here.

Key areas for contribution:
- **Service Layer**: Business logic in `packages/core/src/`
- **Adapter Layer**: Convex integration in `packages/backend/convex/`
- **Frontend Components**: React UI in `apps/web/src/components/`
- **Testing**: Backend services and frontend component tests
