2. Click **"Continue with Email"**
3. Create an account or sign in
4. âœ… **Done!** You now have working authentication

### Step 3: Set Up OAuth (Optional - 3 minutes)

If you want Google and GitHub sign-in:

1. **Google OAuth**:
   - Go to [Google Cloud Console](https://console.cloud.google.com/)
   - Create OAuth 2.0 credentials
   - Add redirect URI: `http://localhost:3003/auth/google/callback`
   - Copy Client ID and Secret to `packages/backend/.env.local`

2. **GitHub OAuth**:
   - Go to [GitHub Developer Settings](https://github.com/settings/developers)
   - Create new OAuth App
   - Add callback URL: `http://localhost:3003/auth/github/callback`
   - Copy Client ID and Secret to `packages/backend/.env.local`
